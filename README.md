# Projeto E-Portal
 
Projeto de TCC do E-Portal, um site de gerenciamento de notícias relacionadas a projetos extracurriculares e cursos da ETEC Poá. O projeto foi feito por um grupo de 5 alunos do 3° ano do ensino médio integrado ao Técnico em Desenvolvimento de Sistemas.
